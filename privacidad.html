Aviso de Privacidad y Confidencialidad

Bolgsvenia Global

Este aviso explica cómo Bolgsvenia Global gestiona la información personal y los documentos enviados para servicios de traducción.

1. Quiénes Somos

Bolgsvenia Global es un servicio de traducción con sede en el Reino Unido que ofrece apoyo de traducción Español ↔ Inglés para particulares, empresas y clientes profesionales.

Para consultas sobre privacidad, puede contactar con Bolgsvenia Global por correo electrónico: bolgsvenia.global@bolgsvenia.com

2. Información que Podemos Recopilar

Cuando contacta con Bolgsvenia Global, utiliza el formulario de contacto del sitio web o solicita un presupuesto de traducción, podemos recopilar información como:

  • Su nombre y datos de contacto, como correo electrónico y número de teléfono si lo proporciona.
  • Información contenida en correos electrónicos, mensajes, consultas o formularios de contacto.
  • Dirección de la traducción, tipo de documento, número de páginas, país donde se utilizará la traducción y fecha límite.
  • Documentos o archivos enviados o adjuntados para presupuesto o traducción.
  • Información contenida en los documentos, como nombres, fechas, direcciones, datos de certificados e información legal, administrativa, académica, empresarial o personal.
  • Información de pago y registros básicos del negocio cuando sea necesario.

3. Cómo Utilizamos su Información

La información personal y los documentos se utilizan únicamente cuando sea necesario para responder a consultas, preparar presupuestos, prestar servicios de traducción y gestionar la administración relacionada.

  • Responder a consultas recibidas por correo electrónico o mediante el formulario del sitio web.
  • Revisar documentos para preparar un presupuesto.
  • Valorar los requisitos de traducción, certificación, formato y plazo.
  • Preparar y entregar traducciones.
  • Comunicarnos con clientes sobre trabajos de traducción.
  • Preparar declaraciones de traducción certificada cuando proceda.
  • Mantener registros básicos de negocio, facturación y pagos.
  • Responder a preguntas, correcciones o solicitudes de seguimiento.

4. Base Legal para el Uso de la Información

Bolgsvenia Global utiliza información personal cuando es necesario para responder consultas, preparar presupuestos, prestar servicios de traducción, conservar registros de negocio y cumplir obligaciones legales o administrativas.

Esto puede incluir el uso de información antes de acordar un servicio, para prestar un servicio de traducción acordado, para cumplir obligaciones legales o para la administración legítima del negocio.

5. Formulario del Sitio Web y Documentos Adjuntos

El sitio web puede incluir un formulario de contacto que permite enviar consultas y adjuntar un documento para fines de presupuesto. Los datos enviados mediante el formulario pueden incluir nombre, correo electrónico, número de teléfono si se facilita, detalles de la consulta, tipo de documento y cualquier archivo adjunto.

Los envíos realizados mediante el formulario del sitio web pueden ser procesados por un proveedor externo de formularios, como Web3Forms, para enviar la consulta a Bolgsvenia Global por correo electrónico.

Se recomienda no enviar documentos altamente sensibles salvo que sea necesario. Para archivos grandes, varios documentos o documentos altamente confidenciales, contacte primero con Bolgsvenia Global para confirmar la forma más adecuada de recibirlos.

6. Confidencialidad

Los documentos y la información personal se tratan de forma confidencial y se utilizan únicamente para preparar presupuestos, prestar servicios de traducción y gestionar registros relacionados.

Bolgsvenia Global no vende información ni documentos de clientes.

7. Compartir Información

La información no se comparte salvo que sea necesario para el servicio de traducción, requerido por ley o acordado con el cliente.

Cuando se requiera una traducción jurada, revisión profesional o apoyo especializado, los documentos pueden compartirse con un traductor jurado autorizado, traductor, agencia o revisor profesional únicamente para ese fin.

La información también puede gestionarse mediante correo electrónico, servicios de formularios web, almacenamiento en la nube, sistemas contables u otras herramientas empresariales utilizadas para administrar el trabajo de traducción.

8. Durante Cuánto Tiempo Conservamos la Información

Los documentos y la correspondencia relacionada se conservan solo durante el tiempo necesario para el servicio de traducción, comunicación con el cliente, solicitudes de corrección, registros de negocio, contabilidad o fines legales y administrativos.

Si desea que sus documentos sean eliminados después de finalizar la traducción, puede solicitarlo por correo electrónico. Algunos registros básicos pueden conservarse cuando sea necesario por motivos comerciales, fiscales, legales o contables.

9. Seguridad

Se toman medidas razonables para proteger documentos e información personal frente a acceso no autorizado, pérdida, uso indebido o divulgación.

No obstante, ningún sistema de comunicación en línea puede garantizar seguridad absoluta. Se recomienda a los clientes no enviar información sensible innecesaria cuando no sea necesaria para el servicio de traducción.

10. Sus Derechos

Según las circunstancias, puede tener derechos conforme a la normativa de protección de datos del Reino Unido, incluido el derecho a solicitar acceso a su información personal, corrección de datos inexactos, eliminación, restricción del tratamiento u oposición a ciertos usos de su información.

Para realizar una solicitud de privacidad, contacte con: bolgsvenia.global@bolgsvenia.com

11. Cookies del Sitio Web

Este sitio web es un sitio estático sencillo. Actualmente no utiliza cookies de análisis, cookies publicitarias ni sistemas de pago en línea propios.

El formulario de contacto puede utilizar servicios externos necesarios para procesar consultas y reducir spam. Si en el futuro se añaden herramientas de análisis, publicidad, sistemas de reserva u otras funciones, este aviso podrá actualizarse.

12. Actualizaciones de este Aviso

Este aviso de privacidad y confidencialidad puede actualizarse ocasionalmente para reflejar cambios en los servicios, herramientas utilizadas, requisitos legales o prácticas de gestión de datos.
